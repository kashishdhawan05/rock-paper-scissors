# Rock Paper Scissors 🎮

A command-line Rock Paper Scissors game built in Python using clean logic and optimized decision-making.

---

## 📌 Features

* User vs Computer gameplay
* Randomized computer choices
* Input validation
* Clean and efficient game logic

---

## ▶️ How to Run

1. Make sure Python is installed
2. Run the following command:

```bash
python main.py
```

---

## 🧠 How It Works

The game compares the user's choice with the computer's choice using a logical pattern instead of multiple conditions, making the code efficient and scalable.
